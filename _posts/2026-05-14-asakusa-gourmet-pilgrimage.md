浅草は<mark>よにのちゃんねる</mark>と<mark>Snow Man（すの日常）</mark>の2グループが、それぞれ独立して何度も選んできたエリアだ。2グループ合わせて6店・10本以上の動画に浅草のお店が登場しており、もんじゃ・すき焼き・おにぎり・うなぎ・麦とろという浅草らしい老舗食文化が揃っている。1エリアで異なるグループの聖地を一気に巡れる、ファンにとって効率のいい聖地だ。

## よにのちゃんねるの浅草【2店】

よにのが浅草で選んだのは、もんじゃと老舗すき焼きという対照的な2店。どちらも「浅草でゆっくり食事を楽しむ」という空気感で収められた回だ。

### てっぱん大吉（もんじゃ）

よにのが2回訪問した「よにの公認聖地」。初登場の<mark>#272</mark>（夜ごはんシリーズ）で3人が鉄板を囲む様子が話題になり、<mark>#490</mark>（忘年会）でも再登場した。1つの店が複数回登場するのはよにの動画でも珍しく、それだけメンバーが気に入っていることの証明でもある。

メンバーが注文した<mark>特選明太スペシャルもんじゃ</mark>と<mark>浅草のりもんじゃ</mark>のセットが動画を再現できる組み合わせ。スタッフが焼き方を丁寧に教えてくれるため初めてでも安心だ。

> **電話予約（03-5826-7021）推奨**。週末・忘年会シーズンは行列必至。**営業時間 11:00〜23:00（火・水は17:00〜）**。つくばエクスプレス浅草駅A1-1出口から**徒歩約3分**。

### 浅草今半 国際通り本店（すき焼き）

<mark>#366</mark>「打ち上げをしてみた日」で3人が訪れた、1895年（明治28年）創業の老舗すき焼き。130年を超える歴史を持ち、浅草随一の格式ある名店だ。打ち上げの舞台にこの店を選んだあたりが、よにのメンバーの食へのこだわりを象徴している。

> **予約必須**（公式サイトまたは電話）。ランチ・ディナーともに営業。**最寄り：東京メトロ銀座線浅草駅 徒歩約5分**。

## Snow Man（すの日常）の浅草【4店】

すの日常が浅草に登場したのは4本。おにぎり・もんじゃ・麦とろ・うなぎと、浅草の食文化を幅広く体験した足跡が残っている。

### おにぎり浅草宿六（おにぎり）

<mark>深澤辰哉・渡辺翔太・向井康二・ラウール</mark>の4人が2024年9月の動画「東京で最古…元祖にぎりめし」で訪れた、1954年創業の東京最古のおにぎり専門店。ミシュランガイドにも掲載されており、注文を受けてから一つひとつ手で握るスタイルが70年変わらない。

> **営業時間 10:00〜18:00（売り切れ次第終了）**。人気のため**午前中の早い時間帯**に並ぶのがベスト。**最寄り：東京メトロ銀座線浅草駅 徒歩約8分**。

### 浅草つる次郎 HANARE（もんじゃ・お好み焼き）

<mark>渡辺翔太・阿部亮平・目黒蓮・佐久間大介</mark>がアスレチック前の腹ごしらえとして2024年5月に訪れたもんじゃ・お好み焼き店。「アスレチックに行く前にまずはメシ！」というタイトルが示す通り、4人でにぎやかに鉄板を囲む様子が収められている。よにのとは別の4人が浅草のもんじゃ文化を楽しんだ回でもある。

### 浅草むぎとろ本店（麦とろ）

雷門から徒歩1分という立地にある麦とろご飯の専門店。すの日常では「むぎとろを食べたい！すのちゅーぶ初の飲酒！？」と題した2026年4月の動画に登場した。とろろと麦ご飯というシンプルな組み合わせを、Snow Manが浅草観光とセットで楽しんだ回。

> **最寄り：東武スカイツリーライン浅草駅 徒歩約2分**。雷門のすぐそば。**11:00〜21:00**営業（土日は〜20:30）。浅草観光の昼食にも使いやすい。

### うなぎ川松（うなぎ）

<mark>岩本照・深澤辰哉・佐久間大介・目黒蓮</mark>が「釣り部結成記念食事会」として2024年2月に訪れたうなぎ専門店。40年以上継ぎ足し続ける秘伝タレで焼くうなぎが名物で、浅草の老舗として地元ファンにも愛されてきた一店だ。「釣りをする → 浅草でうなぎを食う」という流れが、すの日常らしい食への本気度を物語っている。

## グループ別モデルコース

6店を1日で全部まわると食べすぎになるため、グループ別に1日ずつ組むのが現実的だ。

### よにのルート（夕方〜夜）

浅草観光を午前中に済ませ、夕方からよにの聖地へ向かうコースが定番。

| 時間 | 場所 |
|------|------|
| 11:00〜 | 浅草寺・仲見世通りを観光 |
| 14:00〜 | 雷門・駒形橋エリア散策 |
| 17:30〜 | **てっぱん大吉**（#272・#490の聖地。もんじゃで1〜2時間） |
| または | **浅草今半**（#366の打ち上げ気分を再現。要予約） |

### Snow Manルート（朝〜夜）

おにぎり宿六は売り切れが早いため朝イチが鉄則。そのまま浅草観光を挟み、昼・夜と進む。

| 時間 | 場所 |
|------|------|
| 10:00〜 | **おにぎり宿六**（売り切れ前に並ぶ。深澤・渡辺・向井・ラウールの4人が訪れた） |
| 11:30〜 | 浅草寺・雷門エリア観光 |
| 13:00〜 | **浅草むぎとろ本店**（むぎとろでランチ。雷門すぐそば） |
| 16:00〜 | 浅草仲見世・合羽橋道具街散策 |
| 18:00〜 | **うなぎ川松**（釣り部メンバーと同じ秘伝タレのうなぎでディナー） |
| または | **浅草つる次郎**（渡辺・阿部・目黒・佐久間のもんじゃ回を再現） |

> 2日間で両ルートを制覇する「浅草よにの×Snow Man横断巡礼」も可能。浅草寺エリアに宿泊すれば2グループ分の聖地を無理なく回れる。
