松重豊演じる井之頭五郎が一人でランチを楽しむ「孤独のグルメ」。Season10（2022年放送）は相模原・目黒・横浜・埼玉・千葉・岐阜・富山・東京と各地を巡り、インドネシア料理・フレンチ・飛騨郷土料理・広東料理・韓国式中華・イタリア料理など多彩なジャンルが登場しました。

この記事では全12話の登場店舗14店と五郎が実際に注文した料理を完全まとめで紹介します。

## 各話の聖地詳細

### 第1話｜神奈川県相模原市橋本「よしの食堂」
相模原市橋本の昭和から続く大衆食堂。五郎は牛肉のスタミナ炒め定食を注文し、ねぎ玉とカツ煮を追加した。Season10の開幕を飾ったボリューム満点の大衆食堂回。

**注文料理：** 牛肉のスタミナ炒め定食 / ねぎ玉 / カツ煮

{% include inline-shop-card.html shop_id="kodoku_no_gurume-eab54e11-" %}

---

### 第2話（前半）｜東京都品川区目黒「チャベ 目黒店」
目黒のインドネシア料理店。ソトババット（牛もつ煮スープ）から始まり、ナシゴレン・ルンダン（牛肉のスパイス煮）・ミーアヤム（チキンと麺）・シオマイと次々と注文。インドネシア料理の奥深さに五郎が引き込まれた回。

**注文料理：** インドネシア紅茶（アイス） / ソトババット・ミニ / ナシゴレン / ルンダン（牛肉のスパイス煮） / ミーアヤム / トランチャム / シオマイ コンプリート

{% include inline-shop-card.html shop_id="kodoku_no_gurume-193f57bf-" %}

### 第2話（後半）｜東京都高田馬場「もつ焼きばん 高田馬場店」
高田馬場のもつ焼きチェーン。インドネシア料理の後に立ち寄り、もつ焼きとホッピーで締めた。

**注文料理：** もつ焼き / ホッピー

{% include inline-shop-card.html shop_id="kodoku_no_gurume-b768e841-" %}

---

### 第3話｜神奈川県横浜市桜木町「キッチン・カフェ ばる」
横浜・桜木町の魚料理カフェ食堂。黒板日替わりメニューから真鯛のソテー オーロラソースを選び、まぐろのユッケどんぶりといくらとしらすのどんぶりも追加した横浜港を感じる回。

**注文料理：** 真鯛のソテー オーロラソース / まぐろのユッケどんぶり / いくらとしらすのどんぶり（ミニ） / メンチカツ

{% include inline-shop-card.html shop_id="kodoku_no_gurume-aa39fbb9-" %}

---

### 第4話｜埼玉県新座市大泉学園「ビストロ KUROKAWA」
埼玉県新座市大泉学園の本格フランス料理店。サザエとキノコのプロヴァンス風（バゲット付き）でスタートし、野菜のポタージュスープを経て、奥久慈卵の牛タンシチューのオムライスで締めた郊外フレンチ回。

**注文料理：** サザエとキノコのプロヴァンス風（バゲット付き） / 野菜のポタージュスープ（バジル風味） / 牛タンシチューのオムライス

{% include inline-shop-card.html shop_id="kodoku_no_gurume-kurokawa-" %}

---

### 第5話｜千葉県柏市鷲野谷「いづみ亭」
千葉県柏市鷲野谷の大衆食堂。手賀沼のほとりにある地元密着店で、ネギレバ炒め定食に鶏皮餃子・青ネギラーメンを追加した。五郎が「これが食堂の正しい姿」と感じた回。

**注文料理：** ネギレバ炒め定食 / 鶏皮餃子 / 青ネギラーメン

{% include inline-shop-card.html shop_id="kodoku_no_gurume-c40a5794-" %}

---

### 第6話｜岐阜県下呂市上呂「大安食堂」
岐阜県下呂市上呂のJR高山線・上呂駅そばの郷土料理店。飛騨名物の「とんちゃん（味噌ダレ豚バラ肉の鉄板焼き）」と「けいちゃん（味噌ダレ鶏肉の鉄板焼き）」を鉄板で焼き、途中で生卵を加えてすき焼き風に。山深い飛騨の郷土料理に五郎が感動した回。

**注文料理：** とんちゃん（味噌ダレ豚バラ肉の鉄板焼き） / けいちゃん（味噌ダレ鶏肉の鉄板焼き） / 生卵（追加・すき焼き風）

{% include inline-shop-card.html shop_id="kodoku_no_gurume-547030ba-" %}

---

### 第7話｜東京都渋谷区笹塚「山横沢」
東京都渋谷区笹塚の沖縄そば専門店。麩と野菜・ゴーヤの沖縄風炒め「ふーちゃんぷるー」と、トマトカレーつけそばという斬新な組み合わせを注文し、笹塚で沖縄の空気を楽しんだ回。

**注文料理：** ふーちゃんぷるー（麩・野菜・ゴーヤの沖縄風炒め） / トマトカレーつけそば

{% include inline-shop-card.html shop_id="kodoku_no_gurume-f5a8fc06-" %}

---

### 第8話（前半）｜富山県富山市岩瀬浜「アナザーホリデー」
富山県富山市岩瀬浜の海鮮食堂。富山名物「かに面（雌ズワイガニの甲羅おでん）」と海鮮とろろ丼（ハーフ）を注文した富山ロケ前半店舗。

**注文料理：** かに面（雌ズワイガニの甲羅おでん） / 海鮮とろろ丼（ハーフ）

{% include inline-shop-card.html shop_id="kodoku_no_gurume-7134368f-" %}

### 第8話（後半）｜富山県富山市岩瀬浜「居酒屋 舞子」
同じく岩瀬浜の海鮮居酒屋。鯛の昆布締め・ふくらぎの刺身・富山おでん各種・白エビのかき揚げと富山の海の幸を次々と制覇した富山ロケ後半店舗。

**注文料理：** 鯛の昆布締め / 刺身（ふくらぎ） / おでん各種（大根・すすたけ・里芋） / 白エビのかき揚げ / ご飯

{% include inline-shop-card.html shop_id="kodoku_no_gurume-842790b5-" %}

---

### 第9話｜東京都荒川区三河島「世味」
東京都荒川区三河島のコリアタウンにある韓国式中華料理店。韓国式酢豚「タンスユク」と、チャンポンとジャジャン麺のハーフ＆ハーフ「チャムチャ麺」という斬新な一皿に五郎が驚いた日暮里・三河島ロケ回。

**注文料理：** タンスユク S（韓国式酢豚） / チャムチャ麺（海鮮チャンポンとジャジャン麺のハーフ＆ハーフ）

{% include inline-shop-card.html shop_id="kodoku_no_gurume-41d621a0-" %}

---

### 第10話｜神奈川県川崎市武蔵小杉「粥菜坊」
神奈川県川崎市武蔵小杉（元住吉）の食べログ百名店に選ばれる広東料理店。豚肉腸粉（チョウフン）・竹昇麺のワンタンメン・朝鮮人参粥・牡蠣の中華お好み焼きと本格広東料理を堪能した回。

**注文料理：** 豚肉腸粉（チョウフン） / 雲吞麺（ワンタンメン・竹昇麺使用） / 朝鮮人参粥 / 牡蠣の中華お好み焼き

{% include inline-shop-card.html shop_id="kodoku_no_gurume-2fa4ed63-" %}

---

### 第11話｜千葉県旭市「レストラン バイキング」
千葉県旭市のビストロ。千葉のブランド豚「いも豚」を使った「塩わさびの豚ロースソテー300g」が看板メニュー。食後のミルクセーキとともに千葉・旭市の豊かな食材を堪能した回。

**注文料理：** 塩わさびの豚ロースソテー（いも豚300g・野菜添え） / ミルクセーキ

{% include inline-shop-card.html shop_id="kodoku_no_gurume-0ddde1a8-" %}

---

### 第12話（最終回）｜東京都千代田区麹町「ラ・タベルナ」
東京都千代田区麹町の創業45年イタリア食堂。特製ミートローフにバターライス・スパゲッティが添えられたハーフランチセット、さらにニンニクとマヨネーズのスパゲッティ「ミスキアーレ」のハーフセットを追加してSeason10を締めくくった。

**注文料理：** 特製ミートローフ ハーフランチセット（バターライス＆スパゲッティ・ミニサラダ・コーヒー） / ミスキアーレ（ニンニクとマヨネーズのスパゲッティ）ハーフセット

{% include inline-shop-card.html shop_id="kodoku_no_gurume-84b06962-" %}

## Season10の特徴：多彩なジャンルと各地ロケ

Season10は神奈川・埼玉・千葉・岐阜・富山まで足を伸ばした広域ロケシーズン。インドネシア料理・フレンチ・飛騨郷土料理・沖縄そば・韓国式中華・広東料理・イタリア料理と、毎話異なるジャンルが登場しました。特に第6話の「とんちゃん・けいちゃん」（岐阜・下呂）と第8話の富山・岩瀬浜2店はしごが人気を集めました。
