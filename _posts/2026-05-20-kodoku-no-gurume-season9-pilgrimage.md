松重豊演じる井之頭五郎が一人でランチを楽しむ人気ドラマ「孤独のグルメ」。Season9（2021年放送）では神奈川・東京・静岡・群馬・福島と広範囲をロケし、ギリシャ料理・台湾料理・貴州料理・モンゴル料理といった異国の料理が続々登場しました。

この記事では全話の登場店舗と五郎が実際に注文した料理を一覧で紹介します。

## 各話の聖地詳細

### 第1話｜川崎市宮前平「とんかつしお田」
川崎市宮前平にあるとんかつ専門店。五郎はヒレかつ御膳を注文し、魚介クリームコロッケとエビフライを追加。ボリューム満点のとんかつランチを完食した。

**注文料理：** ヒレかつ御膳 / 魚介クリームコロッケ / エビフライ

{% include inline-shop-card.html shop_id="kodoku_no_gurume-6b42b0e8-" %}

---

### 第2話｜神奈川県二宮町「山小屋」
神奈川県二宮町の喫茶店。Season9で最も異色の注文スタイルが話題になった回。メニューからあみだくじでトッピングを決める「五郎オリジナルパフェ」（宇治抹茶・パイナップル・マロンクリーム・コーンフレーク）を注文し、思いがけない組み合わせに驚きつつも楽しんだ。

**注文料理：** 宇治抹茶・パイナップル・マロンクリーム・コーンフレークの特製パフェ

{% include inline-shop-card.html shop_id="kodoku_no_gurume-9dee537b-" %}

---

### 第3話｜東京都港区東麻布「ギリシャ料理 タベルナ ミリュウ」
東麻布にある本格ギリシャ料理店。前菜盛り合わせ（ザジキ・タラモサラタ・メリザノサラタ）から始まり、ドルマーデス・ムサカ・海老のサガナキと次々と注文。食後にはバクラヴァとギリシャコーヒーで締めくくり、五郎が「異国の味」に完全に満足した回。

**注文料理：** 前菜盛り合わせ（ザジキ・タラモサラタ・メリザノサラタ） / ドルマーデス / ムサカ / 海老のサガナキ / バクラヴァ / ギリシャコーヒー

{% include inline-shop-card.html shop_id="kodoku_no_gurume-18c1e256-" %}

---

### 第4話｜東京都府中市「しんせらてぃ」
府中市にある台湾料理店。蒸し鶏のピリ辛和え・カキとニラの辛し炒め・割包（カーポー）・鰻の蒲焼チャーハン・ナスの冷麺と台湾料理を次々と制覇。さらに3倍杏仁とカニあんかけ焼きそばをテイクアウトで追加し、五郎が台湾の多彩な味に夢中になった大食回。

**注文料理：** 蒸し鶏のピリ辛和え / カキとニラの辛し炒め / 割包（カーポー） / 鰻の蒲焼チャーハン / ナスの冷麺 / 3倍杏仁 / カニあんかけ焼きそば

{% include inline-shop-card.html shop_id="kodoku_no_gurume-e476aabd-" %}

---

### 第5話｜静岡県伊東市宇佐美「焼肉ふじ」
静岡県伊東市宇佐美の焼肉店。県外ロケとなった第5話で、五郎は牛焼きしゃぶ（ミスジ）・豚焼きしゃぶ・上ハラミをたっぷり楽しみ、締めに石焼きビビンバへ赤身カルビを乗せた「石焼き赤身カルビビビンバ」を完成させた。海辺の町での豪快な焼肉回。

**注文料理：** 牛焼きしゃぶ（ミスジ） / 豚焼きしゃぶ / 上ハラミ / キムチ / 石焼きビビンバ / 赤身カルビ

{% include inline-shop-card.html shop_id="kodoku_no_gurume-7e51bfa4-" %}

---

### 第6話｜東京都豊島区南長崎「割烹・定食 さがら」
豊島区南長崎の割烹・定食店。肉とナスの醤油炒め定食に鳥唐揚げと冷やし中華を加えた、家庭的でほっとする定食屋回。五郎がごく普通の定食の奥深さを楽しんだ一話。

**注文料理：** 肉とナスの醤油炒め定食 / 鳥唐揚げ / 冷やし中華

{% include inline-shop-card.html shop_id="kodoku_no_gurume-94021473-" %}

---

### 第7話｜東京都葛飾区新小岩「貴州火鍋」
新小岩にある中国・貴州省の郷土料理専門店。貴州前菜三種盛り（ハーフ）・厚揚げの回鍋肉・納豆火鍋と珍しい料理が続き、五郎が「こんな火鍋があるのか」と驚いた回。羊肉を追加してシメまで堪能した。

**注文料理：** 貴州前菜三種盛り（ハーフ） / 厚揚げの回鍋肉 / 納豆火鍋 / 羊肉

{% include inline-shop-card.html shop_id="kodoku_no_gurume-dfde9625-" %}

---

### 第8話（前半）｜群馬県高崎市「オリタ焼まんじゅう店」
群馬県高崎市の焼きまんじゅう専門店。群馬名物の焼きまんじゅうをあんありとあんなしで食べ比べ。もちもちとした生地と独特の甘辛い味噌だれに五郎が驚いた回。同じ第8話の後半では別の店へはしごする二軒構成。

**注文料理：** 焼きまんじゅう（あんあり） / 焼きまんじゅう（あんなし）

{% include inline-shop-card.html shop_id="kodoku_no_gurume-0b59836f-" %}

### 第8話（後半）｜群馬県高崎市「えんむすび」
焼きまんじゅうの後に訪れた高崎市のおむすび店。うに・いか明太・ねぎ味噌のおむすびに鮎の塩焼き・大海老にんにく炒め・なめこの味噌汁と充実のラインナップ。五郎がおむすびを何ラウンドも追加し、大満足のはしご食い回となった。

**注文料理：** うにのおむすび / いか明太のおむすび / ねぎ味噌のおむすび / 鮎の塩焼き / 大海老にんにく炒め / なめこの味噌汁

{% include inline-shop-card.html shop_id="kodoku_no_gurume-563905aa-" %}

---

### 第9話｜福島県郡山市「舞木ドライブイン」
福島県郡山市のドライブイン。東北ロケとなった第9話で、五郎は焼肉定食と和風オイル焼肉定食をダブルで注文。帰り際には店頭販売の名物「ぞうりぱん」をテイクアウトした。

**注文料理：** 焼肉定食 / 和風オイル焼肉定食 / ぞうりぱん（テイクアウト）

{% include inline-shop-card.html shop_id="kodoku_no_gurume-b849d4a4-" %}

---

### 第11話｜東京都豊島区巣鴨「シリンゴル」
巣鴨にある本格モンゴル料理店。モンゴルのミルクティー「スーテーツァイ」でスタートし、甘辛のシリンゴルサンド・塩ゆで骨付き羊肉「チャンサンマハ」・ジャージャー麺・蒸しまん「羊肉ボーズ」・ごま団子と次々と制覇。異国情緒あふれる料理に五郎が感動した回。

**注文料理：** スーテーツァイ（モンゴルミルクティー） / シリンゴルサンド / チャンサンマハ（塩ゆで骨付き羊肉） / ジャージャー麺 / 羊肉ボーズ（蒸しまん） / ごま団子

{% include inline-shop-card.html shop_id="kodoku_no_gurume-275fdbc5-" %}

## Season9の特徴：異国料理回が多いシーズン

Season9はギリシャ・台湾・貴州・モンゴルと異国の料理が4回も登場した、シリーズ屈指のワールドグルメシーズン。また第2話のあみだくじパフェ・第8話の群馬二軒はしごなど、ユニークな食べ方が話題を集めました。
